# 📊 Customer Churn Analysis Dashboard (Excel)

## 📌 Overview
This project focuses on analyzing customer churn using Microsoft Excel.  
The objective is to identify patterns behind customer attrition and provide insights that can help improve customer retention.

## 🎯 Objectives
- Measure overall churn rate
- Analyze churn across subscription types
- Compare customer spending behavior
- Identify high-risk customer segments

## 📂 Dataset Description
The dataset contains the following fields:
- Customer ID
- Age
- Gender
- Subscription Type (Basic, Standard, Premium)
- Monthly Charges
- Tenure (in months)
- Payment Method
- Churn (Yes/No)


## 🛠️ Tools & Techniques Used
- Microsoft Excel  
- Pivot Tables & Pivot Charts  
- Slicers for interactivity  
- Data Cleaning 
- Excel Functions:
  - `IF`
  - `COUNTIF / COUNTIFS`
  - `AVERAGEIF`


## 📊 Dashboard Features
- 📌 KPI: Overall Churn Rate  
- 📊 Churn by Subscription Type  
- 🥧 Customer Distribution by Payment Method  
- 📉 Monthly Charges vs Churn Analysis  
- 🎛️ Interactive Slicers (Gender, Subscription Type)


## 🔍 Key Insights
- Basic plan customers showed higher churn rates  
- Standard plan had better retention and stability  
- Premium customers generated high revenue but also exhibited churn risk  
- Lower-priced plans were associated with quicker drop-offs
